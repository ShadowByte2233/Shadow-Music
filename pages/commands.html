Alle Commands

Eine vollständige Übersicht aller verfügbaren Slash-Commands von Shadow Music. Nutze /help im Bot für eine direkte Übersicht im Discord.

Startseite › Commands

🎵 Musik-Playback

/play  m!play
Spielt einen Song oder eine URL ab
query – Song-Name oder URL
Beispiel: /play Bohemian Rhapsody
/pause
Pausiert die aktuelle Wiedergabe
/resume
Setzt die Wiedergabe fort
/stop
Stoppt die Wiedergabe und verlässt den Voice-Channel
/skip
Überspringt den aktuellen Song
/nowplaying
Zeigt den aktuell gespielten Song an
/volume
Lautstärke einstellen
level – Lautstärke (1–100)
/seek
Springt zu einer bestimmten Position im Song
position – Zeit in Sekunden
/listen
Reagiert auf Discord Voice Messages und spielt sie ab
Beispiel: /listen (dann Voice Message senden)

🎛️ Steuerung / Sound

/equalizer
Passt den Equalizer für personalisierten Sound an
preset – bass / treble / flat / pop / rock
Beispiel: /equalizer bass
/loop
Wiederholt den aktuellen Song oder die Queue
mode – off / song / queue
/shuffle
Mischt die Warteschlange zufällig durch
/recommended
Gibt Song-Empfehlungen basierend auf dem aktuellen Song
/disconnect
Trennt den Bot vom Voice-Channel

🎶 Playlists

/playlist create
Erstellt eine neue persönliche Playlist
name – Name der Playlist
/playlist add
Fügt einen Song zur Playlist hinzu
name – Playlist · song – Song-URL oder Name
/playlist load
Lädt eine Playlist in die Queue
name – Name der Playlist
Beispiel: /playlist load MeinePlaylist
/playlist save
Speichert die aktuelle Queue als neue Playlist
name – Name der neuen Playlist
Beispiel: /playlist save Favorites
/playlist list
Zeigt alle gespeicherten Playlists an
/playlist delete
Löscht eine bestehende Playlist
name – Name der Playlist
/import_playlist
Importiert eine Playlist von Spotify, YouTube oder SoundCloud
url – Spotify-, YouTube- oder SoundCloud-URL
Beispiel: /import_playlist https://open.spotify.com/playlist/...

🛠️ Utilities

/queue
Zeigt die aktuelle Song-Warteschlange an
/clear
Leert die gesamte Warteschlange
/remove
Entfernt einen Song aus der Warteschlange
index – Position in der Queue
/lyrics
Zeigt den Songtext des aktuell gespielten Songs an
/ping
Zeigt die Latenz des Bots an
/help
Zeigt alle verfügbaren Commands und deren Beschreibung
